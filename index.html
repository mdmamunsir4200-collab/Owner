01 // The Developer
About, Skills & Stack
Full-Stack Dev Frontend Expert AI / ML Enthusiast Mobile (Flutter) Open to Work Bangladesh 🇳🇩

Hi, I'm Asraful — a driven, self-taught developer from Bangladesh with a deep passion for building smart, high-quality applications. My journey started with pure curiosity and a blank text editor. What came out the other side is a developer who spans the full width of the modern stack.

I move fast. I'm not the developer who waits for a perfect setup — I adapt, learn, and ship. Whether it's a React frontend, a Django API, a Flutter mobile app, or a TensorFlow model, I pick up new tools in record time and use them to build things that actually matter.

"My main focus is building smart, modern apps with new features — and I get genuinely excited about learning something new every single day."

I'm highly proficient in frontend engineering — from pixel-perfect HTML/CSS/Tailwind to complex React, Angular, and Next.js architectures. On the backend, I'm comfortable with Node.js, Django, FastAPI, and Express. I also have a real interest in AI/ML, working with Sklearn, TensorFlow, Pandas, and NumPy.

Beyond that, I build for mobile with Flutter, work with real-time systems using Socket.IO and WebRTC, automate with Puppeteer, and manage everything through professional DevOps workflows.
